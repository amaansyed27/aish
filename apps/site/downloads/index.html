Install AiSH on Windows, macOS, and Linux.

Release artifacts are published from GitHub Actions. Each release should include the desktop installer and the provider shell binary for that OS.

WIN

Windows

Use the MSI for the desktop app. Use the provider shell binary or terminal setup flow for shell-only testing.

Get MSI
MAC

macOS

Use the DMG or app bundle from the macOS build artifact. Gatekeeper trust requires signing and notarization for public launch.

Get DMG
LIN

Linux

Use DEB first. RPM and AppImage are supported as follow-up release targets.

Get Linux builds
